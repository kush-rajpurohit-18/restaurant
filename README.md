# Restaurant App

A full-stack restaurant application built with Next.js (frontend) and NestJS (backend).

## Project Structure

```
restaurant/
├── fe/   # Next.js frontend
└── be/   # NestJS backend
```

## Getting Started

### Frontend (fe/)

```bash
cd fe
npm install
npm run dev
```

Runs on https://restaurant-ochre-eight.vercel.app/
### Backend (be/)

```bash
cd be
npm install
npm run start:dev
```

## Tech Stack

- **Frontend**: Next.js, TypeScript, Tailwind CSS
- **Backend**: NestJS, TypeScript, Prisma
